# chess-pyxel

Jeu d'échec fait avec pyxel dans le cadre du projet de groupe de NSI du lycée Simone-Veil.

## Installation

```bash
git clone https://github.com/TheDogHusky/chess-pyxel.git
cd chess-pyxel
```

Installez pyxel:

```bash
# windows
python -m pip install pyxel
```

## Exécution

```bash
python src/ui_test.py

```

## Fichiers

- `chess.pyxres` - Fichier de ressources pyxel
- `ui_test.py` - Code incomplet du jeu: seulement l'interface utilisateur et sa logique
